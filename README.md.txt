# RAG Chatbot for NetSol's Financial Report and General Knowledge

This is a Retrieval-Augmented Generation (RAG) chatbot that answers queries related to NetSol's financial report, current events, and general knowledge. The chatbot uses a combination of models, including `Flan-T5` for general questions and a financial document retriever for finance-related queries.

## Features
- **Finance Queries**: Answering questions based on NetSol's financial data.
- **General Knowledge Queries**: Answering general questions using a large language model (Flan-T5).
- **Current Event Queries**: Placeholder for live event updates.
